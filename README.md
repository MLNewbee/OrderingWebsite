## 实训过程与感受
#### 实训过程确实有点累，确实让人内心不安，技术上的不足，任务分配的不均等等，项目开发的压力，都是让人头疼的。但是，正是这种逆人性的经历或者说锻炼是能让人进步最大的。通过这次实训，确实学到了很多东西，也找到自己的兴趣所在。这次实训，我主要负责后端开发。实训是按照正式软件开发流程来进行的，组完队后，立项，配置相关开发环境，安装相关开发软件。我们选择的开发项目是开发一个订货网站。最初的两天我和帆进行文档的编写，最重要的两个文档是需求分析和系统设计。其实，刚开始时有点抵触写这些繁琐的文档的，但是写完文档后发现自己对于项目有了更清晰的理解和全局掌控。所以，磨刀不误砍柴工啊。接下来我们就进行开发，后端选择的技术是Struts2 Java＋JDBC＋MySQL。开发软件是MyEclipse 2017和MySQL软件。实训我主要负责的模块是评论模块，订单模块，后台管理员模块

#### 评论模块有些bug来不及改了所以就流产了。其他模块基本完成。然后对于项目的完成度，直达到了50％，很多功能都没有实现。主要原因是自己懒，拖延症重，工作效率低下，对于技术的探索不够积极。

#### 项目评比的时候，我被别的组的作品给震撼到了，特别是有个小组做的音乐平台，跟 网易云音乐可以相媲美~ 前端和后端都很强大。还是要继续努力的，奋起直追！加油！
